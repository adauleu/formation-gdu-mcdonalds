Application de recherche de restaurants McDonald's grâce à l'API Nominatim

## Pour installer les dépendances
```sh
npm install
```
ou
```sh
yarn install
```

## Pour lancer l'application en local
```sh
npm dev
```
ou
```sh
yarn dev
```

## Pour lancer les tests
```sh
npm test
```
ou
```sh
yarn test
```
